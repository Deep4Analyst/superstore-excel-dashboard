# Super Store Sales Dashboard – Excel

This is a self-driven Excel dashboard project created to analyze Super Store sales data and generate business insights using Pivot Tables, charts, and KPIs.

## Tools Used
- MS Excel
- Pivot Tables
- Charts & Graphs
- Slicers
- Data Cleaning Techniques

## Project Overview
- Built an interactive Excel dashboard to analyze Super Store sales data.
- Calculated key KPIs such as Total Sales, Average Sales, Total Transactions, and Total Quantity.
- Performed region-wise, category-wise, and ship mode-wise sales analysis.
- Analyzed monthly trends to understand demand and seasonality.
- Identified top-performing cities and customer segments.
- Converted raw data into meaningful business insights for decision-making.
